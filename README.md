# 地震岩盤崩壊シミュレータ

六角形グリッドで岩盤の連鎖崩壊をシミュレート。マグニチュードの対数スケールが体感できる教育アプリ

🔗 **公開URL**: https://earthquake.meetupsensei.com/

## 概要

六角形グリッドで表現された岩盤に応力を加え、隣接するブロックへ連鎖的に崩壊が広がっていく様子をシミュレートするアプリです。崩壊の規模からWells & Coppersmith式を用いてマグニチュードを算出し、グーテンベルク・リヒター則（G-R則）や大森則といった地震学の基本法則をグラフで可視化します。「100回自動実行」を使えば、地震の発生頻度とマグニチュードの分布（対数スケール）を直感的に体感できます。

## 使い方

- グリッド上の六角形をクリックして応力を加え、崩壊の連鎖を発生させる
- 「リセット」ボタンでグリッドを初期状態に戻す
- 「▶ 100回自動実行」ボタンで地震発生をまとめてシミュレーションし、統計を確認する
- 計算過程パネルを開いて、マグニチュードの算出手順（Wells & Coppersmith式）を確認する
- タブを切り替えてG-R則・大森則のグラフを表示する
